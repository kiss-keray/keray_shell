# 行为准则

我们希望 Keray Shell Rust 的社区保持友善、专业和高效。无论是提交 issue、参与讨论还是贡献代码，请尊重彼此的时间和经验差异。

## 我们鼓励

- 清晰描述问题、环境和复现步骤。
- 讨论技术观点时聚焦事实、代码和用户影响。
- 对新贡献者保持耐心，帮助他们理解项目上下文。
- 接受不同实现方案，并用可验证的理由推进讨论。

## 我们不接受

- 人身攻击、羞辱、骚扰或歧视性言论。
- 泄露他人隐私信息、凭据或未公开漏洞细节。
- 在讨论中持续刷屏、偏离主题或恶意消耗维护者时间。
- 以威胁、贬低或嘲讽方式推动自己的意见。

## 处理方式

维护者可以删除不当内容、关闭讨论、限制参与者互动，或在严重情况下禁止继续参与项目。若你发现违反行为准则的情况，请通过项目维护者提供的私下渠道联系。
